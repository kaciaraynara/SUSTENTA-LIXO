Projeto Sustenta CE: Mapeamento e Gestão de Lixo Urbano

Nossa Proposta: Aprofundando o problema e a solução

O descarte irregular de lixo em áreas urbanas é um desafio complexo, que vai além da simples limpeza. A falta de um canal de comunicação eficiente entre a população e os órgãos responsáveis pela gestão ambiental, como a prefeitura, causa um ciclo de ineficiência: os problemas não são reportados a tempo, os recursos de limpeza não são alocados de forma estratégica, e o lixo continua a se acumular, gerando problemas de saúde pública, desvalorização de áreas e degradação ambiental.

É por isso que a nossa equipe criou o Sustenta CE. Nosso objetivo não é apenas construir um aplicativo, mas sim uma ponte digital que empodera o cidadão e otimiza a atuação do poder público. A solução oferece uma plataforma de fácil uso para que a comunidade se torne parte ativa da fiscalização, enquanto a gestão municipal ganha uma ferramenta poderosa para agir de forma preditiva e organizada, baseada em dados reais e geolocalizados. Nossa solução busca contribuir diretamente para o Objetivo de Desenvolvimento Sustentável (ODS) 11, tornando nossas cidades mais limpas, seguras e, acima de tudo, mais inteligentes e colaborativas.

Escopo do Nosso Trabalho: Detalhando o que será entregue
Nesta primeira etapa, focamos 100% no planejamento e na arquitetura. O que está no nosso escopo de documentação é o seguinte:

O que faremos (in-scope):

Aplicativo para o Cidadão (Mobile): Um app robusto e intuitivo onde qualquer pessoa pode criar uma conta ou usar sem login (anônimo), ver um mapa de lixo em tempo real, criar novos relatos com foto e geolocalização precisas e acompanhar o status dos seus próprios relatos através de uma lista pessoal.

Painel para o Gestor (Web): Uma plataforma completa para a equipe da prefeitura fazer login seguro, ter uma visão geral (dashboard com métricas-chave), ver a lista detalhada de relatos com filtros avançados e, o mais importante, atribuir os casos a equipes de coleta de forma eficiente.

Backend (API RESTful): A espinha dorsal do nosso sistema, que lida com a lógica de negócio, o armazenamento de dados e a comunicação entre o aplicativo e o painel. A API será desenvolvida seguindo princípios REST para garantir a escalabilidade.

O que não faremos (out-of-scope para a Etapa 2):

Não incluiremos funcionalidades de gamificação, como pontos ou rankings para os usuários, pois o foco inicial é na funcionalidade principal.

Não haverá um sistema de notificações em tempo real. Os usuários precisarão acessar o aplicativo para verificar o status de seus relatos.

Não teremos um módulo de chat integrado entre gestores e cidadãos, pois a comunicação inicial será unidirecional.

Não faremos gráficos e relatórios avançados de análise de dados.

Visão Geral do Sistema: Como as partes se conectam
O Sustenta CE é uma plataforma multiplataforma que se divide em três partes principais, todas conversando entre si através de uma API. Pense assim:

O aplicativo mobile é a ferramenta do cidadão, feita para ser intuitiva e rápida.

O painel web é a central de comando para a gestão pública, feito para organização e eficiência.

O backend é a "inteligência" por trás de tudo, que gerencia a lógica e os dados do sistema.
